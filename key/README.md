# KEY

## 这个工程能学什么
这是一个按键检测实验，在LED工程的基础上加入了按键对象。
他演示了如何用面向对象的方式处理按键输入——按下点亮LED，松开熄灭LED。

## 运行后你会看到什么
- 按下按键，LED点亮
- 松开按键，LED熄灭

## 代码做了什么
主流程是：
1.创建LED对象
2.创建LED的配置选项
3.创建KEY对象
4.创建KEY的配置选项
5.注册LED对象和KEY对象
6.在主循环中获取按键事件，根据事件控制LED

## 建议重点看哪几段
- BSP/KEY下的驱动代码（状态机 + 消抖 + FIFO事件缓冲）
- 主循环中的事件驱动写法

## 你会学到的知识点
- OOP封装输入设备（GPIO输入）
- 按键消抖的软件实现
- 状态机模式处理按键（按下/弹起/长按/连发）
- FIFO事件缓冲，避免事件丢失
- 事件驱动的编程思路
