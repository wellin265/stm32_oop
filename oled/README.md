# OLED

## 这个工程能学什么
这是一个 OLED 显示屏实验，通过 I2C 驱动 SSD1306，用面向对象的方式封装了完整的显示驱动。
他演示了如何在 OLED 上绘制几何图形、显示字符串/中文，以及播放 BMP 位图动画。

## 运行后你会看到什么
- 先显示一组几何图形（直线、矩形、圆），停留 3 秒
- 再显示中文"嵌入式学习"，停留 3 秒
- 最后循环播放一段宇航员 BMP 位图动画

## 代码做了什么
主流程是：
1.创建 OLED 对象
2.创建 OLED 的配置选项（绑定 I2C 句柄和设备地址）
3.注册 OLED 对象
4.调用 draw_xxx 系列函数绘制几何图形，一次性 DMA 刷新
5.调用 show_cn 显示中文，DMA 刷新
6.在主循环中逐帧播放 BMP 位图动画

## 建议重点看哪几段
- BSP/OLED 下的驱动代码（脏页跟踪、Bresenham 直线、Midpoint 圆、UTF-8 中文渲染）
- 主循环中的 BMP 动画播放逻辑
- I2C + DMA 的配合使用

## 你会学到的知识点
- I2C 通信协议（SSD1306 的指令/数据写入）
- 面向对象风格的显示驱动封装
- 脏页跟踪 — 只刷新修改过的区域，大幅减少 I2C 通信量
- DMA 传输 — 释放 CPU，传输期间可处理其他任务
- Bresenham 直线算法和 Midpoint 圆算法（纯整数运算）
- 点阵字体的渲染原理（6×8 / 8×16 / 16×16 中文）
- BMP 位图的逐帧动画播放
