# 这可能是你能搜到的最好的 STM32 OOP 教程！！！

基于每个嵌入式爱好者手中都有的 `STM32F103C8T6`，覆盖大部分外设以及常用的模块。只需进行少量修改就可以用于自己的项目，具备强可移植性和可阅读性。在开启 `-O3` 优化的情况下，OOP 带来的额外开销极小——只多一到两个机器周期（在 72MHz 频率下也就多了几十 ns）。
